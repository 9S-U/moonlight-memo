月灯メモ｜アイコン設定メモ

このセットには、ホーム画面用アイコン設定済みのファイルが入っています。

入っているもの
- index.html
- apple-touch-icon.png
- icon-192.png
- icon-512.png
- favicon.png
- site.webmanifest
- .nojekyll

使い方
1. このフォルダの中身を GitHub のリポジトリ直下に置く
2. GitHub Pages を有効にする
3. 公開URLを iPhone の Safari で開く
4. 共有ボタン →「ホーム画面に追加」
5. 月灯メモのアイコンとして、設定した画像が使われる

メモ
- iPhone のホーム画面用では apple-touch-icon.png が特に大事
- favicon.png はブラウザタブ用
- site.webmanifest は PWA 系の補助
- もし前のアイコンが残る時は、Safari の再読み込みやホーム画面追加のやり直しをすると反映されやすい

月灯メモ｜そのまま差し替えアイコン版

このセットでは、ユーザーが希望した「2枚目の見た目そのまま」を
ホーム画面用アイコンに使う版へ差し替えています。

差し替える主なファイル
- apple-touch-icon.png
- icon-192.png
- icon-512.png
- favicon.png

やること
1. GitHub リポジトリ内の同名ファイルを、このセットのファイルで置き換える
2. Commit する
3. GitHub Pages の反映を待つ
4. iPhone のホーム画面にある月灯メモを一度消す
5. Safari からもう一度「ホーム画面に追加」する

メモ
- iPhone は古い Web Clip アイコンをキャッシュしやすいので、
  再追加のほうが反映されやすいです。
- preview_exact_icon.png は見た目確認用です。

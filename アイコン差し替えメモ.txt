月灯メモ｜余白少なめアイコン版

このセットでは、ホーム画面でアイコンが小さく見えすぎないように、
採用アイコンの透明な外周余白を強めに詰めた版へ差し替えています。

使い方
1. GitHub リポジトリ内の既存ファイルを、このフォルダの同名ファイルで置き換える
2. 特に以下の4つが大事です
   - apple-touch-icon.png
   - icon-192.png
   - icon-512.png
   - favicon.png
3. GitHub Pages 側が更新されたら、iPhone のホーム画面アイコンを一度削除
4. Safari でページを開き直して、もう一度「ホーム画面に追加」

メモ
- iPhoneホーム画面は古いアイコンをキャッシュすることがあるので、
  再追加のほうが反映されやすいです。
- preview_tight_icon.png は見た目確認用です。
